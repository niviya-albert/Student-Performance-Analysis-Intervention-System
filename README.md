# Student Performance Analysis & Intervention System

A Python-based system to analyze student performance and provide actionable insights.

## 📌 Overview  
This project is a simple Python-based system built to understand student performance using basic data analysis techniques.  
The goal is to identify patterns in student data and highlight students who may need additional attention.

## 🎯 What this project does  
- Analyzes student performance using factors like attendance, CGPA, and test preparation  
- Classifies students into *High, Medium, and Low risk categories*  
- Compares performance across different courses  
- Provides simple suggestions to help improve performance  

## 🛠️ Tech Used  
- Python  
- Pandas  
- Matplotlib  

## 📊 Dataset  
The dataset used in this project is simulated, but designed to reflect real-world academic scenarios.  
It includes:  
- Attendance percentage  
- Past CGPA  
- Test preparation status  
- Course-wise scores  

## ⚙️ How it works  
The system processes student data step-by-step:  
1. Organizes and reads the dataset  
2. Analyzes performance metrics  
3. Identifies best and lowest performing courses  
4. Classifies students based on performance  
5. Generates insights and basic improvement suggestions  

## 📈 Output  
- Class performance summary  
- Course-wise comparison  
- Risk classification of students  
- Visual graphs for better understanding
